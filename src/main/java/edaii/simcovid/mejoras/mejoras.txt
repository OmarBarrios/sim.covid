- Nombre: Mascarilla
- Descripción: Una persona sana o inmune puede tener mascarilla (esto no disminuye la probabilidad de contagio).
- Miembros añadidos a VirusParameters: probabilityOfSuccessWithMasc
- Como se muestra: emoji con mascarilla.


- Nombre: Muerte
- Descripción: Solo una persona infectada tiene probabilidad de morir.
- Miembros añadidos a VirusParameters: probabilityOfSuccessDead
- Como se muestra: emoji con mascarilla.


- Nombre: Rodeado
- Descripción: Si una persona sana tiene a su alrededor infectados.
- Miembros añadidos a VirusParameters: probabilityOfSuccessDead
- Como se muestra: emoji con mascarilla.

 ROWS
 COLUMNS
 MSECONDS_PER_DAY

 DAYS_TO_BE_INMUNE                  = number of days that have to pass to become immune
 PROBABILITY_OF_SUCCESS_INFECTED    = 0.0 to 0.9
